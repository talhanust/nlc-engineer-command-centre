2026-06-15 — Workspace power-user layer + OSM basemaps

  persisted via the workspace key, live during drag).

  collapses, expands full-page, and docks.

  expand the panel under the cursor. Typing targets are ignored.

  or breakdown to a sticky rail that stays put while the page scrolls; clears
  on navigation. Hidden under 1100px.

- Persistence: theme + last-visited node remembered; app reopens where you
  left off (HomeRedirect → last node).

  distribution mix, and the receipts/payments/liabilities tables.
- Maps: Leaflet + OpenStreetMap made first-class — basemap switcher on every
  map and a default-basemap setting (Auto/OpenStreetMap/Light/Dark); OSM uses
  the canonical tile.openstreetmap.org tiles. Theme now drives tiles directly.
  Tests +6 → web 200 passing; server 8; typecheck clean; both builds green.

2026-06-15 — Workspace interactivity (sidebar / focus mode / zoom)
- Collapsible sidebar: header toggle slides it away with a width/opacity
  transition and reclaims the space; state persisted (nlc-ecc.workspace). On
  narrow screens it becomes an off-canvas drawer with a click-away scrim.
- Focus mode: new reusable Focusable (render-prop) puts an Expand affordance on
  panels and opens them full-page in an overlay (Esc/✕ to close, background

  Full-screen maps enable scroll-wheel zoom + pan via MapView interactiveZoom.
- Content zoom: header A−/100%/A+ control scales the whole content area
  (0.8–1.4, persisted); click the value to reset.
- Fullscreen: header button toggles the browser Fullscreen API for a control-
  room view.
- State: UiState gained persisted sidebarOpen + zoom. Tests +3 (Workspace).
  Web 194 passing, server 8; typecheck clean; both builds green.

  Date of Completion, plus optional Client and a map location picker. New
  projects default to 0% progress.

  the project executive tab. Removed the old static PakistanMap (SVG logic moved
  into domain/geo.ts).
- Data: createProject persists code/dates/coords; updateProject handles the new
  fields; new updateNodeLocation provider method (both providers). api mode
  needs no server/DB change — the new fields ride in the existing /api/state
  JSONB docs.
- Tests: +9 (domain/geo, data/projectCreation). Updated ProjectLifecycle UI test
  to the new field set. Web 191 passing, server 8 passing; typecheck clean;
