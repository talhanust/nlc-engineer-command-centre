2026-06-16 — Fix clipped post titles + show occupant names on the chart

2026-06-16 — Editable organogram (HR staff edit on the chart)
- The organogram is now an editing surface. An "Edit organogram" toggle on the
  Organogram tab turns on inline affordances: ＋ add a post under any box, ✎ edit
  (title / scale / category / auth / held / reports-to), ✕ remove (with confirm;
  orphans re-parent to the deleted box's parent). An "Add section" tile and an
  "Add head post" empty-state cover the structural cases.
- Re-parenting two ways: the editor's Reports-to dropdown and drag-and-drop —
  drag any box onto another to re-assign its parent, with a cycle guard so a box
  can't be dropped onto its own descendant.
- A lightweight UnitEditorPanel (inline, not a modal) handles add/edit; every
  change persists immediately via upsertHrUnit/deleteHrUnit and the chart
  re-renders. Editing operates on the authored establishment; if the tier was
  still category-derived, entering edit first materialises real posts so edits
  stick. Export buttons follow the edited set.
- Tests +1 (add a section via the chart) → web 225; server 8; typecheck clean;
  both builds green.
